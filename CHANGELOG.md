# 7.0.0-rc.9 (2025-11-21)

كيف تستخدم هذا الملف؟
بما أنك قمت بإنشاء الملف، يمكنك الآن التحكم بمشروعك كخدمة احترافية باستخدام الأوامر التالية في التيرمنال الخاص بالخادم (VPS):

لبدء تشغيل المشروع للمرة الأولى:

Bash
pm2 start ecosystem.config.js
لمراقبة حالة السيرفر واستهلاك الرامات والمعالج في الوقت الفعلي:

Bash
pm2 monit
لعرض سجلات التشغيل (Logs) لمعرفة ما يحدث خلف الكواليس:

Bash
pm2 logs whatsapp-api
لإيقاف أو إعادة تشغيل المشروع:

Bash
pm2 stop whatsapp-api
pm2 restart whatsapp-api
